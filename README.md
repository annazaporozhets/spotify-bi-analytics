# Spotify Analytics Dashboard

## Description 
This project investigates key success factors in the modern music industry using Spotify data. By analyzing audio features like energy, danceability, and tempo, the study explores how technical characteristics influence audience reach and track popularity, while uncovering patterns in music consumption and content performance.

## Project Overview
This project involved developing interactive analytical dashboards to explore factors driving track popularity on Spotify. The analysis covers key audio features and user preferences, providing actionable insights into industry trends.

## Key Insights & Findings
* Audio Features Analysis: The dashboard visualizes the relationship between track energy, danceability, and explicit content, highlighting distinct clusters of track characteristics.
* Trend Identification: Uncovered patterns in track duration relative to popularity, showing how different track lengths perform in the current music market.
* Tool Comparison: The project served as a comparative study between Tableau and Power BI, evaluating their efficiency in data visualization and cross-filtering capabilities for business reporting.
